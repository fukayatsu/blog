# 今回のリリース内容

Webhookの設定で、 `on post update (only WIP to ShipIt)`にチェックを入れると記事がWIP => ShipItに切り替わったタイミングのみWebhook通知をすることができます。

※ ShipItした記事も、一旦WIPに戻した後にShipItすると初回ShipItとして扱われます。
